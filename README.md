# Visão geral

Projetos desenvolvidos em NodeJs, com o intuito de **estudar** e se **aprofundar** na linguagem, com projetos reais.
<br>

<p id="start"></p>

### - Pré-requisitos globais:

`npm install -g nodemon typescript ts-node`

### - Instalação:

`npm install`

### - Rodar no server local:

`npm start`
